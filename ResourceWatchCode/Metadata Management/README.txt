Updating Sources - can be used to ensure that all source organization names and
subtitles are consistent throughout the RW platform's Source Organization and
Subtitle metadata fields.

Extension - search the description, limitations, and function fields to ensure
that these fields are also consistent with the Official RW names.

https://docs.google.com/spreadsheets/d/1ijQrjYv-mxtmoDHxv0S1XlfRed6IbYcviRW_HY7GAh8/edit?usp=sharing

Upload Metadata to API - can be used to bulk upload metadata from the RW
metadata sheet to the RW API. Requires a valid user token to authorize changes.
