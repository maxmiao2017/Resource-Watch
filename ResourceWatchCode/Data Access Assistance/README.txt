A collection of scripts intended to help using the various data platforms
and data types that the RW API utilizes.

* Google Earth Engine - a brief catalogue of commands for interacting with
data on GEE
* NetCDF - a brief catalogue of commands for interacting with data in the
NetCDF format
* RW Data Package Creator - a brief catalogue of commands that can be used to
subset vector and raster data to a given bounding box
* rw-api - a python package available on PyPA
(pip install rw-api-tools, from rw-api-tools.rw-api-tools import rw-api-tools,
rw = rw-api-tools() )
